# Секции сайта

Все секции внутри одного `index.html`, идут в этом порядке:

1. **Hero** (`<header class="hero">`) — имя, роль, CTA на Telegram и форму
2. **Обо мне** (`#about`) — два абзаца о карьерном пути и инфостиле
3. **Опыт** (`#experience`) — таймлайн из 4 этапов: Нейромедиа → Конверт → автор Unisender → фриланс
4. **О чём пишу** (`#topics`) — две колонки тем (нейросети / маркетинг)
5. **Избранные материалы** (`#works`) — 3 карточки-ссылки на реальные статьи + кнопки на профили автора
6. **Где публикуюсь** (`#publications`) — сетка логотипов площадок: Unisender, Нейромедиа, vc.ru, Хабр, TexTerra, SendPulse, SMMplanner, Ак Барс банк, Sendsay. Логотипы подтягиваются через `https://www.google.com/s2/favicons?domain=X&sz=128` — ключи и аккаунты не нужны.
7. **Канал** (`#channel`) — большая карточка с Telegram-иконкой (инлайн SVG)
8. **Связаться** (`#contact`) — плашка с email редакции + форма (имя / email / сообщение)
