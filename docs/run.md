# Как запускать локально

Лучше — через live-server. Он следит за файлами и перезагружает вкладку сам:

    npx --yes live-server --port=5173 --no-browser --quiet
    open http://localhost:5173

Альтернатива — открыть `index.html` двойным кликом. Тогда без автообновления, но никаких зависимостей не нужно.

Сборки нет и не планируется — любые изменения в `.html`, `.css`, `.js` попадают в браузер сразу после сохранения.

## Тесты

В проекте есть автоматические тесты для функций-утилит (сейчас — только валидация длины сообщения). Запуск:

    npm test

Выводит список проверок и итог «N passed». Если что-то сломано — пункт будет помечен `not ok` с описанием.
