# Форма обратной связи и контакты

## Как работает форма сейчас

Чисто клиентская. На `submit` скрипт `js/contact-form.js` блокирует стандартную отправку, скрывает форму и показывает блок «Спасибо». **Никуда ничего реально не отправляется** — сообщения теряются.

## Если пользователь попросит «сохранять сообщения»

Не выбирать решение молча — обсудить варианты:

- **Formspree / Web3Forms / Getform** — бесплатные тарифы, форма остаётся HTML, меняется только атрибут `action` и поле `_replyto`
- **Свой бэкенд** — overkill для портфолио, упоминать только если пользователь сам этого хочет
- **Google Forms embed** — если нужна полноценная анкета, а не «напишите мне»
