# Спецификация: рабочая форма обратной связи

Версия 2.0. Первая версия предполагала VPS + свой Node-бэкенд — отказались, т. к. сайт делается для примера и покупать домен + сервер ради демо-формы нерационально.

## Обзор архитектуры

Полностью статическая сборка:

- **Хостинг:** GitHub Pages — бесплатно, из того же репо `mikeshu2001/portfolio`, URL вида `https://mikeshu2001.github.io/portfolio/`. HTTPS автоматический.
- **Обработка формы:** [Formspree](https://formspree.io) — бесплатный тариф, 50 сообщений в месяц. Форма на сайте отправляет данные напрямую к ним, они пересылают их на email.
- **Своего бэкенда нет.** Никакого VPS, Nginx, Node.

## Что видит посетитель

1. Заполняет форму (имя / email / telegram / сообщение) и жмёт «Отправить».
2. Кнопка блокируется на время запроса и меняет текст на «Отправка…».
3. На успех форма скрывается, показывается блок:
   > **Спасибо!**
   > Сообщение отправилось. Отвечу в течение пары дней.
4. На ошибку (сеть упала, Formspree недоступен) над формой появляется строка:
   > Не удалось отправить. Попробуйте ещё раз или напишите на **neiro@unisender.com** напрямую.
   Форма остаётся заполненной, кнопка разблокируется, посетитель может повторить.

## Поля формы

| Поле      | Тип      | Обязательное | Атрибуты HTML                            |
| --------- | -------- | ------------ | ---------------------------------------- |
| Имя       | text     | да           | `autocomplete="name"`                    |
| Email     | email    | да           | `autocomplete="email"`                   |
| Telegram  | text     | нет          | placeholder «@username — по желанию»     |
| Сообщение | textarea | да           | `rows="5"`                               |

## Защита от спама

Два уровня, бесплатно:

1. **Honeypot через поле `_gotcha`.** Formspree распознаёт это имя как ловушку: скрытое поле, которое человек не заполнит, а бот — заполнит автозаполнением. Любая отправка с непустым `_gotcha` молча отбрасывается.
2. **Встроенный спам-фильтр Formspree.** Анализирует IP, частоту, подозрительные паттерны.

CAPTCHA пока не ставим. Если этих двух уровней перестанет хватать — добавим Cloudflare Turnstile (бесплатно, невидимый).

## Куда улетает сообщение

**Только email на `shymovskaia73@gmail.com`.**

Formspree автоматически формирует письмо со всеми полями формы. Тема вида «New submission from portfolio» (можно настроить в админке Formspree). В теле — список «поле: значение».

**Telegram-уведомления отложены.** Для их реализации нужен либо платный тариф Formspree ($10/мес) плюс Zapier-прокладка, либо переход на Cloudflare Worker. Для demo-проекта оверкилл. Если форма начнёт активно использоваться — возвращаемся к этому.

## Автоответ отправителю на email

**Не реализуем.** На бесплатном тарифе Formspree автоответов нет. Роль подтверждения берёт на себя страничный блок «Спасибо».

## Техническая сторона

- **Отправка:** JS-обработчик перехватывает submit, собирает `FormData`, делает `fetch` к Formspree endpoint с заголовком `Accept: application/json`. Formspree отвечает JSON → показываем нужный блок на странице.
- **Атрибут `action`** у `<form>` указывает на Formspree endpoint (вида `https://formspree.io/f/xxxxxxx`) — нужен даже при JS-отправке, т. к. JS его оттуда читает.
- **Состояние кнопки:** блокировка + текст «Отправка…» на время запроса, разблокировка на ошибке.
- **Валидация:** `required` + `type="email"` на HTML + `form.checkValidity()` в JS. Серверная валидация — дело Formspree.

## Чек-лист подготовки (что нужно от пользователя)

- [ ] **Включить GitHub Pages** в репо: настройки → Pages → Source: `main` branch → `/ (root)` → Save. Через минуту сайт доступен на `https://mikeshu2001.github.io/portfolio/`.
- [ ] **Зарегистрироваться на [formspree.io](https://formspree.io/register)** (можно через GitHub-аккаунт).
- [ ] **Создать новую форму** в Formspree, указав email получателя `shymovskaia73@gmail.com`. Сервис выдаст endpoint вида `https://formspree.io/f/xxxxxxx`.
- [ ] **Первая отправка** — Formspree пришлёт письмо подтверждения на `shymovskaia73@gmail.com`: надо кликнуть по ссылке «Confirm email», чтобы форма заработала.
- [ ] **Передать endpoint** в чат — я подставлю его в HTML одной правкой.

## Что оставлено на потом

- **Telegram-уведомления.** Отложены до реального спроса, план — Cloudflare Worker (бесплатно).
- **Согласие на обработку ПД.** В коде стоит заглушка-комментарий `<!-- TODO: checkbox + политика -->`. Если сайт будет собирать заметный трафик из РФ/РБ, вернёмся.
- **Автоответ отправителю.** Только с платным Formspree или своим бэкендом. Пока хватает успеха на странице.
- **Свой домен.** Бесплатно не получить, но можно купить потом и привязать к GitHub Pages через CNAME.
