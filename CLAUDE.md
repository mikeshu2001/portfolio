# Портфолио Михаила Шумовского

Одностраничный сайт-визитка. Пользователь — **Михаил Шумовский**, главный редактор «Нейромедиа» (проект Unisender), **не разработчик**. Работает с прикладными нейросетями и цифровым маркетингом.

Стек: чистый HTML/CSS/JS без сборки и зависимостей. Не предлагать React, Next, Tailwind, TypeScript и т. п. без явной просьбы.
